# Real-Time AI System on Raspberry Pi

Proyecto desarrollado para el curso de Sistemas en Tiempo Real.

## Hardware

- Raspberry Pi 3 B+
- Cámara USB

## Tecnologías

- Linux Embebido
- POSIX Threads (pthreads)
- Lenguaje C
- Git y GitHub

## Objetivo

Desarrollar un sistema en tiempo real para:

- Captura periódica de imágenes
- Preprocesamiento
- Inferencia mediante una red neuronal
- Control de un actuador

El proyecto será desarrollado de forma incremental.
